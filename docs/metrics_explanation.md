# Metrics Explanation

## Precision

Precision tells how many predicted detections are correct.

Formula:

Precision = True Positives / (True Positives + False Positives)

## Recall

Recall tells how many real objects were detected.

Formula:

Recall = True Positives / (True Positives + False Negatives)

## mAP

mAP means mean average precision. It summarizes object detection performance across classes.

## Confusion Matrix

A confusion matrix shows where the model predicts the correct class and where it confuses one class with another.
