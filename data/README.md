# PPE Dataset Overview

This directory is intended to store the dataset used to train and evaluate the PPE safety violation detector.

The project does not include dataset images by default. Download an appropriate dataset with annotations for:

- `person`
- `helmet`
- `no_helmet`
- `safety_vest`
- `no_vest`

Recommended dataset sources:

- Roboflow Universe PPE / construction safety datasets.
- Ultralytics Construction PPE dataset.

Download the dataset in YOLO format and follow `dataset_instructions.md`.
